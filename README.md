# 🔧 Balken-Auslegungstool

Interaktives Web-Tool zur Berechnung und Visualisierung von Biegemomenten, 
Querkräften und Sicherheitsfaktoren für verschiedene Balkenkonfigurationen.

## Funktionen
- 5 Lastfälle: Streckenlast, Einzellast, Kragarm, Einzellast an beliebiger Position
- Biegemomentenlinie und Querkraftverlauf als Diagramm
- Materialauswahl (S235, S355, Aluminium)
- Sicherheitsfaktor Berechnung mit Ampel-System

## Technologien
- Python, NumPy, Matplotlib, Streamlit

## Starten
pip install streamlit numpy matplotlib
streamlit run balken_app.py
